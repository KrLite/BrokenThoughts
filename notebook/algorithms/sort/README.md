# 排序算法

排序算法分为稳定排序和不稳定排序[^稳定性]。稳定排序是指排序后能够保证**相同元素**的**相对位置**不变，不稳定排序则不能够保证。

[^稳定性]: 如果排序算法中存在**交换元素**的操作，那么该排序算法就**可能**是不稳定排序。
